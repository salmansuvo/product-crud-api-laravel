# API_VUE Laravel-8

## Project setup
```
composer update
```

### Generate APP:KEY
```
php artisan key:generate
```

### Run Migrations
```
php artisan migrate
```

### Seed Data into Database
```
php artisan db:seed
```

### Generate JWT secret key
```
php artisan jwt:secret
```
